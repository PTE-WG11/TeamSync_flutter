## 📋 文档信息
- **项目名称**: TeamSync 团队协作管理系统
- **版本**: v1.0
- **平台**: Flutter Web (桌面端)
- **日期**: 2026-02-10

---

## 1. 设计概述

### 1.1 设计理念

**青灰森林系 (Teal & Slate)** - 区别于市面上泛滥的蓝紫 SaaS 工具，采用蓝绿色（Teal）为主色，传递"高效、理性、自然生长"的团队协作感，同时降低视觉疲劳。

### 1.2 视觉层次

```
┌─────────────────────────────────────────┐
│           主要内容区 (Surface)            │
│  ┌─────────────────────────────────┐   │
│  │     卡片层级 (Card)              │   │
│  │   ┌─────────────────────────┐   │   │
│  │   │    悬浮元素 (Elevated)   │   │   │
│  │   └─────────────────────────┘   │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
         ↓              ↓              ↓
    Background      Surface       Elevated
    #F8FAFC        #FFFFFF       +Shadow
```

## 2. 颜色系统

### 2.1 主色调 (Primary Colors)

| 名称 | 色值 | 用途 |
|------|------|------|
| **Primary** | `#0D9488` | 主按钮、进度条、选中状态、链接 |

| **Primary Hover** | `#0F766E` | 按钮悬停状态 |
