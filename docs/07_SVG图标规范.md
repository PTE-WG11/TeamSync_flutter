# TeamSync SVG 图标规范

## 📋 文档信息
- **项目名称**: TeamSync 团队协作管理系统
- **版本**: v1.0
- **平台**: Flutter Web (桌面端)
- **日期**: 2026-02-10

---

**1. 图标视觉风格**
系统统一采用线性（Outline）风格，强调简洁与专业感。
- **线条特征**：线宽固定为 1.5px（基于 24px 画布），所有线条端点和连接处均采用圆角处理。
- **造型标准**：图标带有 2px 的圆角，整体建立在 24x24px 的基础网格上，核心内容通常限制在 20x20px 的视觉范围内。

**2. 尺寸应用体系**
规范定义了五个标准尺寸，以适应不同的界面层级：
- **XS (16px)**：用于极小图标或内联文字。
- **SM (20px)**：适用于小按钮或紧凑列表。
- **MD (24px)**：默认尺寸，用于大多数导航菜单和操作项。
- **LG (32px)**：用于强调型按钮或卡片图标。
- **XL (48px)**：专用于空状态展示或大型统计图标。

**3. 资源组织与分类**
图标资源按功能严格分类存储于 `assets/icons/` 目录下，包括：
- **导航（Navigation）**：首页、项目、日历、设置等。
- **任务（Task）**：子任务、标记、优先级等。
- **状态（Status）**：规划中、进行中、已完成、逾期等。
- **操作（Action）**：增删改查、筛选、排序、更多等。
- **文件与反馈**：上传下载、附件、以及成功/错误/警告等反馈提示。

**4. Flutter 技术实现**
- **核心依赖**：统一使用 `flutter_svg` 插件进行矢量图渲染。
- **代码规范**：SVG 源码需保持整洁，移除多余属性，描边颜色统一使用 `stroke="currentColor"` 以便在代码中动态着色。
- **封装组件**：通过 `AppIcon` 组件统一调用，支持通过枚举值 `IconSize` 控制尺寸，并内置了点击反馈（InkWell）支持。
- **常量管理**：所有图标路径均在 `AppIcons` 类中定义为常量，避免在业务代码中硬编码字符串。
